Algoritmo

Aprenda a programar entendendo o seu funcionamento


O que é um Algoritmo?

Algoritmo é um conjunto de instruções a serem seguidas em sequência. Exemplo: instrução nº 1, instrução nº 2, etc.

Exemplo do dia a dia

Lâmpada queimada:
Passo 1: Aproximar a escada
Passo 2: Girar a lâmpada no sentido inverso e substituí-la
Passo 3: Descer da escada e clicar o interruptor

Exemplo com programação

Passo 1: Abrir o prompt (MS-DOS)
Passo 2: Digitar node.js e pressionar ENTER
Passo 3: Digitar mkdir taskmanager
Passo 4: Digitar cd
Passo 5: Digitar mkdir src
Passo 6: Digitar install express

Conclusão

Um algoritmo é curto e finito. Um sistema é constituído de vários algoritmos.

Quem entende isso consegue programar qualquer sistema, pois todos seguem o mesmo princípio.
